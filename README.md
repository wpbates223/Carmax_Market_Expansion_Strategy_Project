# CarMax Market Expansion Strategy Analysis

This is a personal project using a small dataset that analyzes market opportunities and infrastructure planning for CarMax expansion using Python.

## Business Questions
- Where are the largest opportunity areas to reach new customers?
- What store format best serves each market?
- What is the expected return on investment (ROI)?
- How will online sales shift infrastructure needs?
- Which regions require additional reconditioning centers?

## Project Structure

data/
Market and demographic datasets

src/
Python analysis modules

## Methods
Market Opportunity Scoring
Store Type Optimization
Financial ROI Modeling
Reconditioning Capacity Planning
